# NURI 프로젝트 제품 요구사항 명세서 (PRD)

## 1. 개요

NURI는 사회적 영향력을 가진 글로벌 스마트팜 스타트업으로, 장애인 고용 통합과 스마트 농업 기술을 결합한 비즈니스를 전 세계로 확장하는 것을 목표로 합니다. 본 문서는 NURI의 웹 플랫폼 구축을 위한 제품 요구사항을 정의합니다. 이 플랫폼은 회사의 글로벌 확장 현황, 시장 진출 단계, 그리고 각 국가별 데이터를 시각적으로 보여주는 대시보드 역할을 합니다.

## 2. 목표

*   **투자자 및 파트너 대상 홍보**: NURI의 글로벌 비전과 실제 성과를 효과적으로 전달하여 투자 유치 및 파트너십 기회를 확대합니다.
*   **데이터 시각화**: 복잡한 시장 및 농장 데이터를 직관적인 3D 인터랙티브 지도를 통해 쉽게 이해할 수 있도록 제공합니다.
*   **글로벌 정체성 강화**: 다국어 지원을 통해 NURI가 글로벌 기업임을 명확히 합니다.

## 3. 사용자 페르소나

*   **투자자 (Venture Capitalist)**: NURI의 시장 확장 전략, 잠재력, 현재 성과를 빠르게 파악하고 싶어 합니다.
*   **정부 및 NGO 파트너**: NURI의 사회적 영향력과 특정 국가/지역에서의 활동을 구체적으로 확인하고 싶어 합니다.
*   **내부 경영진 및 직원**: 회사의 전체적인 현황을 모니터링하고 전략적 의사결정에 활용합니다.

## 4. 핵심 기능 요구사항

### 4.1. 3D 인터랙티브 지구본
- WebGL 기반의 3D 지구본을 웹 페이지 중앙에 렌더링해야 합니다.
- 지구본은 자동으로 천천히 회전해야 하며, 사용자는 마우스로 회전, 확대/축소 등 상호작용이 가능해야 합니다.

### 4.2. 시장 등급 시스템
- 각 국가는 3단계의 시장 등급(Tier 1, 2, 3)으로 분류되어야 합니다.
  - **Tier 1 (진출 완료)**: NURI 스마트팜이 이미 운영 중인 국가.
  - **Tier 2 (우선 순위)**: 다음 진출 목표인 중점 국가.
  - **Tier 3 (장기 목표)**: 장기적으로 진출을 고려하는 국가.
- 각 등급별로 지정된 HSL 색상 코드를 사용하여 지구본 위에 국가별 색상을 명확히 구분하여 표시해야 합니다.

### 4.3. 스마트팜 위치 마커
- Tier 1 국가 내에 실제 운영 중인 NURI 스마트팜의 위치를 노란색의 발광(Pulsing) 효과가 있는 마커로 표시해야 합니다.
- 마커에 마우스를 올리면 해당 팜의 이름(예: K1) 또는 간단한 정보가 표시되어야 합니다.

### 4.4. 국가 데이터 툴팁
- 사용자가 특정 국가에 마우스를 올리면 해당 국가의 주요 데이터가 포함된 툴팁이 나타나야 합니다.
- 툴팁에는 국가명, GDP, 인구, 그리고 다른 국가와 비교할 수 있는 핵심 지표가 포함되어야 합니다.

### 4.5. 다국어 지원
- 플랫폼의 모든 텍스트 콘텐츠는 한국어와 영어를 모두 지원해야 합니다.
- 사용자는 언어 전환 기능을 통해 쉽게 언어를 변경할 수 있어야 합니다.

## 5. 비기능적 요구사항

### 5.1. 성능
- 3D 인터랙션은 초당 60프레임(FPS)을 목표로 하여 부드러운 사용자 경험을 제공해야 합니다.
- 페이지의 최대 콘텐츠풀 페인트(LCP) 시간은 3초 미만이어야 합니다.

### 5.2. 접근성
- 웹 콘텐츠 접근성 지침(WCAG) 2.1 AA 레벨을 준수해야 합니다.
- 모든 시각적 요소에는 스크린 리더 등을 위한 대체 텍스트가 제공되어야 합니다.

### 5.3. 보안
- 모든 통신은 HTTPS를 통해 암호화되어야 합니다.
- XSS, CSRF 등 기본적인 웹 취약점으로부터 안전해야 합니다.

### 5.4. 반응형 디자인
- 데스크톱, 태블릿, 모바일 등 다양한 화면 크기에서 콘텐츠가 올바르게 표시되고 상호작용이 가능해야 합니다.
