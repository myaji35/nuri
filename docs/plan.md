# NURI 프로젝트 실행 계획 (plan.md)

## 1. 프로젝트 목표

NURI의 글로벌 스마트팜 비즈니스 현황을 시각적으로 보여주는 인터랙티브 3D 웹 플랫폼을 성공적으로 구축한다. 이를 통해 투자 유치, 파트너십 강화, 그리고 내부 전략 수립을 지원한다.

## 2. 프로젝트 범위

### In-Scope (포함 범위)
- PRD에 명시된 모든 핵심 기능 (3D 지구본, 시장 등급, 위치 마커, 툴팁, 다국어 지원)
- TRD에 정의된 기술 스택을 사용한 개발
- 반응형 웹 디자인 (데스크톱, 태블릿, 모바일)
- Vercel 및 AWS/GCP를 통한 배포

### Out-of-Scope (제외 범위)
- 관리자용 데이터 편집/입력 대시보드 (초기 데이터는 스크립트로 로드)
- 사용자 회원가입 및 로그인 기능
- 상세 분석 또는 리포팅 기능

## 3. 프로젝트 단계 (BMAD 프레임워크 기반)

### Phase 1: 분석 (Analysis) - 완료
- **산출물**: `prd.md`
- **내용**: 프로젝트의 비전, 목표, 핵심 요구사항이 정의되었습니다.

### Phase 2: 기획 (Planning) - 완료
- **산출물**: `trd.md`, `plan.md`
- **내용**: 기술 스택, 아키텍처, API 명세, 데이터베이스 스키마 등 기술적 요구사항이 정의되었고, 본 실행 계획이 수립되었습니다.

### Phase 3: 솔루션 설계 (Solutioning) - 진행 중
- **산출물**: `lld.md`
- **내용**: 프론트엔드 컴포넌트, 백엔드 API 로직, 상태 관리 등 세부적인 저수준 설계가 진행 중입니다. 이 단계가 완료되면 본격적인 개발에 착수합니다.

### Phase 4: 구현 (Implementation) - 예정
- **방식**: 스프린트 기반의 애자일 개발
- **내용**: 아래 정의된 마일스톤에 따라 기능을 개발, 테스트, 통합합니다.

## 4. 상위 레벨 로드맵 및 마일스톤

### 마일스톤 1: Supabase 백엔드 및 데이터베이스 구축 (Sprint 1)
- **목표**: 프론트엔드 개발에 필요한 모든 데이터 기반과 API를 Supabase를 통해 마련합니다.
- **주요 작업**:
  - Supabase 프로젝트 생성 및 기본 설정
  - Supabase 데이터베이스에서 PostGIS 확장 활성화
  - Supabase 대시보드를 사용하여 `countries`, `market_tiers`, `sites` 테이블 스키마 생성
  - CSV 파일 임포트 또는 스크립트를 통해 초기 데이터 로딩
  - 각 테이블에 대한 읽기 전용 RLS(Row Level Security) 정책 설정
  - Next.js 프로젝트에 Supabase 클라이언트 연동 확인

### 마일스톤 2: 프론트엔드 3D 지구본 기반 구축 (Sprint 2)
- **목표**: 기본적인 3D 지구본을 렌더링하고 상호작용의 기틀을 마련합니다.
- **주요 작업**:
  - Next.js 프로젝트 설정 및 Three.js 라이브러리 통합
  - 기본 3D 씬(Scene), 카메라, 조명 설정
  - 자동 회전하는 지구본 렌더링
  - 마우스를 통한 회전, 확대/축소 기능 (OrbitControls) 구현

### 마일스톤 3: 데이터 연동 및 인터랙션 구현 (Sprint 3)
- **목표**: 백엔드 API와 프론트엔드를 연동하여 동적 데이터를 시각화합니다.
- **주요 작업**:
  - API 호출 로직 구현 (SWR/React Query 사용)
  - 시장 등급(Tier)에 따른 국가별 색상 적용
  - 스마트팜 위치에 마커 표시 (Pulsing 효과 포함)
  - 국가 호버 시 툴팁에 동적 데이터 표시 (Raycasting)

### 마일스톤 4: UI/UX 개선 및 최종화 (Sprint 4)
- **목표**: 완성도를 높이고 배포를 준비합니다.
- **주요 작업**:
  - 다국어 지원 (react-i18next) 적용
  - 반응형 디자인 최종 점검 및 최적화
  - 로딩 상태, 에러 처리 등 UI/UX 개선
  - 성능 최적화 및 최종 테스트
  - Vercel, AWS/GCP에 배포
