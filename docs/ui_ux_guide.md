# NURI 프로젝트 UI/UX 디자인 가이드

## 1. 디자인 철학

NURI 플랫폼의 UI/UX는 **신뢰성, 전문성, 따뜻한 기술**이라는 세 가지 핵심 가치를 시각적으로 전달하는 것을 목표로 합니다. 전체적으로 현대적이고 깔끔한 디자인을 채택하여 사용자가 복잡한 데이터를 쉽게 이해하고, NURI의 비전에 몰입할 수 있도록 합니다.

- **모던 & 클린**: 충분한 여백과 정돈된 레이아웃을 사용하여 깨끗하고 전문적인 인상을 줍니다.
- **직관적 인터랙션**: 사용자가 별도의 학습 없이도 3D 지구본과 같은 핵심 기능을 쉽게 조작할 수 있도록 직관적인 인터페이스를 제공합니다.
- **정보 중심**: 가장 중요한 데이터와 메시지가 명확하게 드러나도록 시각적 계층 구조를 설계합니다.

## 2. 색상 팔레트

| 역할 | 색상 | HEX 코드 (추정) | 사용처 |
| --- | --- | --- | --- |
| **Primary** | White | `#FFFFFF` | 배경, 카드, 기본 바탕 |
| **Secondary** | Dark Blue | `#0B1A2D` | 푸터, 어두운 배경 섹션, 강조 텍스트 |
| **Accent** | Bright Yellow | `#FFC700` | 주요 버튼(CTA), 아이콘, 하이라이트 요소 |
| **Text** | Black / Dark Gray | `#111827` | 본문 텍스트 |
| **Subtle Gray**| Light Gray | `#F3F4F6` | 카드 테두리, 구분선, 부가 정보 텍스트 |

## 3. 타이포그래피

전체적으로 가독성이 높은 산세리프(Sans-serif) 폰트 계열을 사용합니다. (예: Pretendard, SUIT 또는 시스템 기본 폰트)

- **H1 (메인 헤드라인)**: `font-bold`, `48px`
  - 예: "데이터 기반의 글로벌 혁신..."
- **H2 (섹션 제목)**: `font-bold`, `36px`
  - 예: "지속 가능한 사회적 영향 창출"
- **H3 (카드 제목)**: `font-semibold`, `20px`
  - 예: "NURI 팜", "IT 솔루션"
- **Body (본문)**: `font-normal`, `16px`
- **Button Text**: `font-semibold`, `16px`

## 4. 컴포넌트 디자인 (shadcn/ui 기반)

### 4.1. Buttons
- **Primary (주요)**: `bg-accent` (노란색 배경), `text-secondary` (어두운 텍스트), `rounded-full` (완전 둥근 모서리). 호버 시 약간 어두워지는 효과.
  - 예: "RAHD BS", "투자 제안서 다운로드"
- **Secondary (보조)**: `bg-transparent`, `border-gray`, `text-secondary`. 호버 시 배경색이 연하게 채워지는 효과.
  - 예: "기술 당하기"
- **Tertiary (텍스트)**: 밑줄이나 배경 없이 텍스트와 화살표 아이콘으로 구성.
  - 예: "모두 보기 →"

### 4.2. Cards
- `bg-white`, `border-subtle-gray`, `rounded-lg`, `shadow-md`.
- 내부에 아이콘, 제목, 설명 텍스트가 구조적으로 배치됩니다.
- 호버 시 그림자 효과가 미세하게 강해져 입체감을 줍니다.

### 4.3. Input Fields
- `bg-white` 또는 연한 회색 배경, `rounded-md`, `border-subtle-gray`.
- 포커스 시 테두리 색상이 `accent` 또는 `primary` 색상으로 변경됩니다.
- 푸터의 입력 필드는 어두운 배경에 맞춰 디자인됩니다.

### 4.4. Navigation
- 상단에 고정(Sticky)되는 헤더입니다.
- 좌측에는 로고, 우측에는 네비게이션 링크와 버튼을 배치합니다.
- 스크롤 시 배경색이 반투명한 흰색으로 변경되어 콘텐츠와 구분됩니다.

## 5. 레이아웃 및 간격

- **Grid System**: 12-컬럼 그리드 시스템을 기반으로 콘텐츠를 정렬하여 일관성과 안정감을 줍니다.
- **Spacing**: 컴포넌트와 섹션 사이에 충분한 여백(Whitespace)을 사용하여 시각적 편안함과 가독성을 높입니다. `margin`과 `padding`은 4의 배수(8px, 16px, 24px, 48px...)를 기준으로 설정합니다.
- **Layout**: 사용자의 시선 흐름을 고려하여 위에서 아래로 자연스럽게 이어지는 단일 페이지 스크롤(Single Page Scroll) 구조를 가집니다.

## 6. 핵심 경험 (UX)

### 3D 인터랙티브 지구본
- **기본 상태**: 자동으로 천천히 회전하여 동적인 느낌을 줍니다.
- **사용자 인터랙션**: 마우스 드래그로 회전, 휠로 확대/축소가 가능해야 합니다.
- **호버 효과**: 국가에 마우스를 올리면 해당 국가의 경계가 하이라이트되고, 색상이 미세하게 밝아지며, 관련 정보가 담긴 툴팁이 나타납니다.
- **마커**: 스마트팜 위치를 나타내는 마커는 미세하게 깜빡이는(Pulsing) 애니메이션 효과를 주어 사용자의 주목을 유도합니다.
